## Build Setup

``` bash
# Set up virtual env
$ virtualenv .

# Activate virtual env
$ source bin/activate

# Install Dependencies
$ pip install -r requirements.txt

# Run server
$ export FLASK_APP=app.py flask run

Make sure to have your MySQL Database running on Port 8889 using MAMP
